# Landing Page — Blindex Premium SP

Landing page de alta conversão para serviços de fechamento de área em vidro Blindex, desenvolvida como projeto de portfólio.

## Visualizar o site

https://landingpagewordpress.netlify.app/

## Tecnologias

- HTML5, CSS3, JavaScript puro
- Sem frameworks ou dependências externas
- 100% responsivo (mobile, tablet, desktop)

## Seções

- Hero com imagem de fundo e CTAs para WhatsApp
- Vídeo institucional
- Serviços oferecidos
- Carrossel de projetos realizados
- Avaliações do Google
- Seção de confiança (loja física)
- Rodapé com endereço completo
- Botão flutuante de WhatsApp
